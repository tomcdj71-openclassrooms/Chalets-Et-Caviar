Provides real estate functionality for the WordPress Real Home Theme.

== Description ==

Crucial Real Estate is a dynamic and feature rich WordPress Plugin to integrate property & agent listings on websites of real estate agents and companies. It brings a free theme and quick demo setup feature altogether!
Visit (https://aarambhathemes.com/plugins/crucial-real-estate) for more information.

Visit [demo](https://demo.aarambhathemes.com/real-home/) of Crucial Real Estate plugin native Real Home theme.

== Main features ==

* Agent Support
* Agent details with related properties, social profiles, contact form
* Customizable property types, statuses, features, locations
* Customizable property labels (Hot Deal)
* Listing frontend management ( property slider, property featured listings, property types, property locations) with native Real Home theme
* Photo gallery and video (YouTube, Vimeo, SWF File and MOV File)
* Floor plan details with accordion toggle
* Property agent details with contact detail as well as contact form
* Related Property listings
* Social sharing (Facebook, Twitter)
* Listing all properties

== PRO Features ==

* Agency support
* Agency details with related agents, social profiles, contact form
* Contact Agency/Agent
* Submit property via Back-End and Front-End
* Unlimited Gallery, Video, Floor Plan, Additional details
* Unlimited PDF attachments
* Print Properties
* User registration with Email Alerts.
* Email Listing Alerts
* Supported Google Captcha
* Social Login, Social Share
* Multiple Listing Layouts

== Installation ==

**From your WordPress dashboard**

1. Visit 'Plugins > Add New',
2. Search for 'Crucial Real Estate' and install the plugin.
3. Activate 'Crucial Real Estate' from your Plugins page.

== Frequently Asked Questions ==

= Does this plugin support any WordPress theme? =

Crucial Real Estate includes support for the Real Home WordPress Theme.
